# Proyecto_IA_Trabajo_2030

# Impacto de la IA en el empleo hacia 2030

Proyecto de análisis donde exploro cómo podría afectar la automatización impulsada por IA a distintos puestos de trabajo para el año 2030.

## Contenido

- `data/AI_Impact_on_Jobs_2030.csv`: dataset simulado con 3.000 puestos.
- `notebooks/01_eda_ia_empleo_2030.ipynb`: análisis exploratorio en Python.
- `powerbi/dashboard_ia_trabajo_2030.pbix`: dashboard interactivo en Power BI.
- `src/`: scripts auxiliares (si corresponde).

## Tecnologías

- Python (pandas, matplotlib, etc.)
- Power BI (modelado, DAX, visualizaciones)

## Preguntas que responde

- ¿Qué tipos de trabajos están más expuestos a la automatización?
- ¿Cómo influye el nivel educativo y los años de experiencia?
- ¿Qué profesiones combinan alto salario con alto riesgo?
